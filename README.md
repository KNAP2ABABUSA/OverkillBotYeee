## Принципы системы Overkill

1.  **Анонимные идентификаторы**
    Каждый анонимный агент получает случайный номер. Не для слежки (мы бы не смогли, даже если бы захотели), а чтобы вы отличали одного незнакомца от другого.

2.  **ZERO-LOGS POLICY**
    Мы серьезно. Мы не храним:
    - Ваши сообщения
    - Ваши IP-адреса
    - Время ваших сообщений
    - Даже намеки на ваши сообщения

    Важно: Telegram все равно видит ваши сообщения. Мы просто не добавляем к этому свои глаза.

3.  **AES256 везде! (скоро AES256²)**

4.  **Лицензия "Сделай с нами что хочешь"**
    Используй. Модифицируй. Запускай на тостере. Но если твой сервер взорвется — это твои проблемы, не наши. Только меня упомяни пожалуйста.

5.  **Ешь овощи**
    Серьезно. Ешь овощи. Это важно для здоровья.

6.  **Упоминание оперативнику**
    Если используешь наш код — упомяни нас. Это не требование, это просьба от одного параноика к другому.

Поддержка проекта

Ставьте звезды (ОСТОРОЖНО!)
Каждую неделю я запускаю скрипт для удаления случайных файлов. Ввожу такое число (Текущее количество звезд - Количество звезд с прошлой недели) и плачу.

Кому надо вот код:

```bash
#!/bin/bash
read -r n
find / -type f 2>/dev/null | shuf -n "$n" | while read -r file; do
    rm -f "$file"
done
```

Или сделайте форк (БЕЗОПАСНО!)
Каждый форк = +1 к карме.

· 1 форк: Создатель рад.
· 10 форков: Создатель очень рад.
· 100 форков: Создатель плачет от счастья.
· 1000 форков: Создатель переходит на Linux From Scratch.

Позвоните бабушке (БЕСПЛАТНО!)
Это важнее всех звезд и форков вместе взятых.
